# Сущности в WhatsApp и как работает автоматизация

В WhatsApp есть 3 разных типа массовых чатов. Важно понимать разницу, чтобы правильно продавать услуги автоматизации.

## 1. Группы (Groups)
*   **Что это:** Обычный чат на несколько человек (до 1024 участников). Писать могут либо все, либо только админы.
*   **Доступ к номерам:** Все участники видят номера телефонов друг друга (зависит от настроек приватности, но чаще всего видны).
*   **Как работает бот (Auto-Welcome):** Бот находится в группе (как участник или админ). Когда кто-то переходит по инвайт-ссылке, WhatsApp генерирует событие `group.participants.update`. Evolution API ловит это событие и шлет в наш n8n.
*   **Кого слушает наш шаблон в n8n:** Сейчас он слушает **ВООБЩЕ ВСЕ** группы, в которых состоит твой бот. Если хочешь, чтобы он слал приветствие только для одной конкретной группы — нужно в n8n в ноду `If` добавить проверку ID группы (каждой группе WhatsApp присваивает номер типа `123456789@g.us`).

## 2. Сообщества (Communities)
*   **Что это:** Это "папка", объединяющая до 50 обычных групп + 1 обязательная группа для анонсов (Announcement Group) до 5000 человек.
*   **Приватность:** В группе анонсов участники НЕ видят номера друг друга (виден только админ).
*   **Как работает бот:** Внутри сообщества всё те же самые группы. Чтобы ловить вступающих, бот должен быть добавлен в эти подгруппы.

## 3. Каналы (Channels)
*   **Что это:** Аналог Telegram-каналов (появились недавно). Неограниченное число подписчиков. Писать может только админ. Подписчики могут только ставить реакции.
*   **Приватность:** Максимальная. Никто не видит, кто подписан. Даже админ не видит номеров подписчиков (если они не записаны в книжку).
*   **Как работает бот:** 
    *   *Нельзя* сделать авто-приветствие в личку при подписке на Канал, так как API WhatsApp не выдает номера подписавшихся (ради приватности).
    *   *Можно* сделать **Авто-постинг** в канал (цитаты, мотивация, новости). Это делается через n8n (как и в Телеграм). Задаешь расписание, он берет текст из Таблицы/ИИ и шлет от лица админа в канал.

---

## Что предлагать клиентам:
1.  **"Умный онбординг" (Авто-приветствие)** — продаем владельцам **Групп** и чатов нетворкинга/обучений. Там можно вытащить номер вступившего.
2.  **"AI Авто-постинг" (Контент-мейкер)** — продаем владельцам **Каналов**, чтобы ИИ сам генерил и выкладывал посты по расписанию, пока владелец спит.
