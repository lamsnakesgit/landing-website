# Руководство по визуалу и генерации лиц (Face Swap)

## Проблема со встроенными моделями диффузии
При использовании стандартных моделей генерации (например, Gemini Image Flash, Midjourney, DALL-E) с референсным изображением лица:
- Модель **перерисовывает** лицо с нуля на основе пикселей исходника.
- Отсутствует точный контроль над "весом" (схожестью) черт лица.
- На крупных планах (close-ups) нейросеть начинает "додумывать" детали, из-за чего пропорции и черты могут искажаться (теряется 100% сходство).
- Возникают проблемы с точностью передачи тона кожи (часто делает светлее, если не задан жесткий промпт на смуглый оттенок).

## Решение для 100% сходства (Face Swap)
Для достижения идеального переноса лица без искажения его геометрии и формы на любых, даже крупных планах, необходимо использовать специализированные инструменты:
- **IP-Adapter / PulseID / Roop / Reactor** (в связке со Stable Diffusion).
- Эти инструменты накладывают лицо 1 в 1 поверх сгенерированного персонажа.

## Требования к промптам для базовых диффузионных моделей
Если Face Swap недоступен, для минимизации искажений в промпт следует добавлять жесткие ограничения:
- `THE FACE MUST BE AN EXACT MATCH to the person in the FIRST reference image`
- `including EXACT SKIN TONE, exact eye shape, exact facial structure, and exact ethnicity`
- `Do not alter her facial features or lighten/darken her skin`
- Для рук: `SHORT NAILS, no long extensions`
- Для тона: явно указывать `naturally tanned, slightly swarthy skin tone (warm olive or light brown, not overly dark)`

- Требуется переход на специализированное API или локальную модель для оставшихся крупных планов и исправления тех, где сходство недостаточное.
