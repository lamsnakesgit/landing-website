# Шаблоны сообщений для WhatsApp рассылки

## Вариант 1 — Прямой результат (АКТИВНЫЙ)
```
Привет, {name}!

Привожу клиентов для бизнеса без вложений в рекламу.

Мои клиенты получают заявки уже в первую неделю — без Яндекс.Директа, без таргета, без SEO.

Интересно узнать как? Просто ответьте на это сообщение
```

## Вариант 2 — Вопрос-крючок
```
Привет, {name}!

Вопрос: а если бы вы могли получать 10-30 новых клиентов в месяц без бюджета на рекламу — было бы интересно?

Я как раз этим занимаюсь. Ответьте "да" — расскажу подробнее
```

## Вариант 3 — Короткий и дерзкий
```
Привет, {name}!

Нахожу клиентов для бизнеса. Без рекламы и холодных звонков.

Если актуально — ответьте, скину кейс
```
