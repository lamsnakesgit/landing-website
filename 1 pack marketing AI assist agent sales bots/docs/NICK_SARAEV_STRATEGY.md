# Стратегия Outreach «Nick Saraev Style» (Ник Сараев)

## Основные принципы

Ник Сараев (Nick Saraev) пропагандирует **гиперперсонализацию** и **автоматизацию**. Главная идея: сообщение не должно выглядеть как массовая рассылка. Оно должно выглядеть так, будто вы потратили 15 минут на изучение профиля человека.

### 1. Этапы воронки (Pipeline)
1. **Lead Scraping:** Поиск целевых контактов (LinkedIn, Instagram) через Google CSE или Apify.
2. **Data Enrichment:** Сбор всей доступной информации:
    - Последние 3-5 постов (о чем пишет?).
    - Описание компании (Tavily AI / Perplexity).
    - Раздел "О себе" (Bio/About).
3. **AI Analysis:** GPT-4 анализирует собранные данные.
    - *Промпт:* "На основе этих постов и био, определи тон общения человека, его текущие интересы и боли".
4. **Message Generation:** Создание сообщения по схеме:
    - *Hook:* Упоминание конкретной мысли из последнего поста ("Кстати, классная мысль в твоем посте про ИИ-агентов...").
    - *Value:* Как твое решение помогает его конкретной боли/цели.
    - *CTA:* Легкий вопрос (не продажа).
5. **Multi-channel:** Если нет ответа в LinkedIn — почта или WhatsApp.

## Что мы внедрим в проект v5

### 1. Обогащение данных (Enrichment)
Для WhatsApp основным ключом является **номер телефона**.
- **Поиск в соцсетях:** Использование инструментов (типа Search4Faces или API-агрегаторов) для нахождения Instagram/LinkedIn по номеру.
- **Tavily AI Исследование:** Автоматический поиск в Google информации о человеке по номеру или ФИО + компании.

### 2. Скрапинг постов (Instagram/LinkedIn)
- Использование **Apify Actors** для получения последних постов из профиля.
- Анализ текста постов через ИИ для выделения ключевых тем.

### 3. Smart Personalization Prompts
Обновление нашего Apps Script или создание Node.js прослойки для генерации сообщения:
```javascript
// Пример промпта для ИИ
var prompt = `Напиши первое сообщение в WhatsApp.
Контакт: ${name}.
Интересы из постов: ${interests}.
Темы последних постов: ${latest_posts}.
Цель: Предложить решение по привлечению клиентов без рекламы.
Тон: Дружелюбный, профессиональный, БЕЗ спама.
Правило: Начни с короткого комплимента или упоминания его поста.`;
```
