# 📊 Выгрузка данных из WhatsApp (Чаты и История)

Инструкция по получению списка всех контактов, групп и истории переписки для анализа и рассылок.

---

## 1. Получение списка всех чатов
Чтобы понять, кому и куда можно писать, нужно выгрузить список чатов (личные и группы).

### Через n8n (Рекомендуется)
Я создал шаблон: `n8n_templates/whatsapp_data_export.json`.
1. Импортируй его в n8n.
2. Запусти ноду **"Fetch All Chats"**.
3. Он выдаст список всех JID (идентификаторов), имен и типов чатов.

### Прямой API запрос (Curl):
```bash
curl -X GET https://evolutionapi.aiconicvibe.store/chat/fetchChats/{{instance_name}} \
  -H "apikey: {{api_key}}"
```

---

## 2. Выгрузка истории сообщений
Полезно для анализа интересов клиента или контекста перед ответом AI.

### API запрос:
```bash
curl -X GET "https://evolutionapi.aiconicvibe.store/chat/fetchMessages/{{instance_name}}?number={{phone_number}}" \
  -H "apikey: {{api_key}}"
```
*Параметр `number` должен быть в формате `79991234567` (без плюса).*

---

## 3. Куда сохранять данные?
Для удобного управления рекомендую связку:
1. **Google Sheets:** Создай таблицу с колонками `Name`, `JID`, `Type`, `Status`.
2. **n8n:** Настрой воркфлоу, который раз в час обновляет эту таблицу.
3. **Анализ:** Теперь ты видишь всю свою базу в одном месте и можешь проставлять статусы (например, "Холодный", "Горячий", "Не писать").

---

## 4. Как понять, кому отправлять?
В воркфлоу рассылки используй ноду **"Filter"**:
- Условие: `Status` равно `Рассылка`.
- Это гарантирует, что ты не напишешь лишним людям.
