# WhatsApp CRM-Рассыльщик — Документация

## Что это

Google Apps Script для рассылки WhatsApp сообщений через Evolution API, прямо из Google Sheets. Работает как мини-CRM: теги, сегменты, воронка продаж, отчёты, автопроверка ответов.

---

## Архитектура

**Один скрипт — две роли:**

1. **`createCRMTemplate()`** — создаёт структуру (3 листа с заголовками, валидацией, примерами). Запускается **один раз** при настройке новой таблицы.

2. **Остальные функции** — работают по готовой CRM-таблице (рассылка, дожим, проверка ответов, таймер).

```
┌─ Google Sheets ────────────────────────────┐
│                                            │
│  📋 Лист "CRM"      ← база контактов      │
│  📊 Лист "Отчёты"   ← лог рассылок        │
│  💬 Лист "Ответы"   ← кто ответил         │
│                                            │
│  Меню: 🚀 Рассылка ──┐                    │
│    ▶️ Рассылка        │                    │
│    🔄 Дожим           │ ← запускаются      │
│    🏷 По тегу         │    из меню         │
│    🔍 Проверить       │    таблицы         │
│    ⏰ Таймер          │                    │
│    📋 Создать CRM ────┘                    │
│                                            │
│           │ (API вызовы)                   │
│           ▼                                │
│  Evolution API (ВПС)                       │
│  https://evolutionapi.aiconicvibe.store     │
└────────────────────────────────────────────┘
```

## Версии скриптов

| Файл | Версия | Что умеет |
|:---|:---|:---|
| `evolution_sender_v2_working.gs` | v2 | Базовая рассылка, статусы |
| `evolution_sender_v3_crm.gs` | v3 | + Spintax, теги, сегменты, follow-up |
| `evolution_sender_v4_full.gs` | **v4 (актуальная)** | + Отчёты, таймер, проверка ответов, 3 листа |

**Используй `v4`** — она включает всё из v2 и v3.

## Быстрый старт (5 минут)

### 1. Создай новую Google Таблицу
### 2. Расширения → Apps Script → вставь код из `evolution_sender_v4_full.gs`
### 3. Удали лишние файлы (оставь один скрипт)
### 4. Сохрани (Ctrl+S) → вернись в таблицу → обнови (F5)
### 5. Меню «🚀 Рассылка» → «📋 Создать CRM» — создаст 3 листа
### 6. Заполни лист «CRM» номерами, поставь статус «отправить»
### 7. «🧪 Тест 1 номера» — проверь связь
### 8. «▶️ Рассылка» — поехали!

## Структура CRM-листа

| Колонка | Что | Пример | Примечание |
|:---|:---|:---|:---|
| A — Телефон | Номер в любом формате | `+7 999 123 4567` | Скрипт сам нормализует |
| B — Имя | Имя контакта | `Анна` | Подставляется в `{name}` |
| C — Компания | Название | `ООО Логистик` | Опционально |
| D — Статус | Текущий статус | `отправить` | **Выпадающий список** |
| E — Теги | Кто он тебе | `клиент, партнёр` | Можно несколько через `,` |
| F — Сегмент | Что ему нужно | `рассылки` | Выпадающий список |
| G — Воронка | Этап продажи | `новый` | Выпадающий список |
| H — Заметки | Свободный текст | `С конференции` | |
| I — Посл. касание | Дата отправки | `12.03.2026 10:20` | Заполняется автоматически |
| J — Прочитано | Статус чтения | | Заполняется при проверке |

### Допустимые значения

**Статус (D):** `отправить` → `отправлено ✅` → `дожим` → `ответил` / `пропустить` / `ошибка ❌`

**Воронка (G):** `новый` → `отправлено` → `ответил` → `КП отправлено` → `сделка` / `отказ`

**Сегмент (F):** `рассылки` / `ии-бот` / `автоматизация` / `всё`

## Функции (все кнопки меню)

### ▶️ Рассылка
Отправляет сообщения **всем строкам со статусом «отправить»**. Обновляет статус, воронку, дату. Spintax делает каждое сообщение уникальным.

### 🔄 Дожим
Отправляет follow-up **только тем, кто «отправлено ✅»** (не ответили). Статус меняется на «дожим».

### 🏷 По тегу
Спрашивает тег (например «клиент»), и рассылает **только контактам с этим тегом**.

### 🔍 Проверить ответы
Опрашивает Evolution API — кто ответил. Записывает ответ в лист «Ответы», меняет статус на «ответил».

### ⏰ Таймер
- **Включить** — каждый час отправляет батч из 10 сообщений (настраивается через `BATCH_SIZE`)
- **Выключить** — останавливает

### 👁 Авто-проверка ответов
- **Включить** — каждые 30 минут проверяет ответы
- **Выключить** — останавливает

### 📋 Создать CRM
Создаёт 3 листа: CRM, Отчёты, Ответы — с заголовками, цветами, валидацией.

## Как менять/дополнять

### Изменить текст сообщения
В скрипте найди `MESSAGE_FIRST` и `MESSAGE_FOLLOWUP`. Формат Spintax:
```
{вариант1|вариант2|вариант3}
```
Скрипт случайно выбирает один. Добавляй свои варианты через `|`.

### Добавить новый тег/сегмент/статус
В функции `createCRMTemplate()` найди `requireValueInList([...])` и добавь значение в массив.

### Изменить частоту таймера
В `enableTimer()` замени `.everyHours(1)` на `.everyMinutes(30)` или `.everyHours(2)`.

### Изменить размер батча
Переменная `BATCH_SIZE = 10` вверху скрипта.

### Изменить паузу между сообщениями
Строка `Utilities.sleep(8000 + Math.floor(Math.random() * 7000))` — пауза 8-15 сек. Меняй числа.

### Добавить новую колонку
1. Добавь в объект `COL`: `NEW_COL: 11`
2. Добавь заголовок в `createCRMTemplate()`
3. Используй `sheet.getRange(i, COL.NEW_COL)` в коде

### Подключить другой инстанс (номер WhatsApp)
Замени `INSTANCE = "8326 301 n8n wa 1"` на имя нового инстанса из Evolution API.

## Безопасность и лимиты

| Параметр | Значение | Зачем |
|:---|:---|:---|
| Пауза между сообщениями | 8-15 сек (рандом) | Защита от бана |
| Батч таймера | 10 сообщений/час | Не спамить |
| Spintax | Каждое сообщение уникальное | WhatsApp не детектит как спам |
| Без ссылок | Нет URL в сообщениях | Ссылки = бан |
| Колонка last_sent | Дата последней отправки | Антиспам/контроль |
