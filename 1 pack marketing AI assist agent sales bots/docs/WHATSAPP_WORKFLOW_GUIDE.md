# 🤖 Инструкция по WhatsApp Автоматизации

## 1. Постинг в Каналы и Чаты
Я подготовил шаблон: `n8n_templates/whatsapp_posting_workflow.json`

### Как импортировать:
1. Зайди в свой n8n: https://n8n.aiconicvibe.store
2. Нажми **Workflows** → **Add Workflow**.
3. Нажми три точки (меню) справа сверху → **Import from File**.
4. Выбери файл `whatsapp_posting_workflow.json`.

### Как это работает:
- **Webhook:** Принимает данные (текст сообщения и ID чата).
- **Evolution API:** Отправляет сообщение через твой подключенный номер.
- **Config:** Здесь уже прописан твой API ключ. Не забудь поменять `87...` на реальное имя твоего инстанса из Evolution Manager.

## 2. Рассылка по статусам (знакомым)
Для этого мы будем использовать связку **Google Sheets (или база данных) + n8n**.

### Логика:
1. В таблице список номеров и их статус (например: "Знакомый", "Клиент").
2. n8n раз в день проверяет таблицу.
3. Если статус подходит — отправляет персонализированное сообщение.
