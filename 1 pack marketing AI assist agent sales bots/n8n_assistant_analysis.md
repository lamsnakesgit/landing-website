# Разбор n8n AI-Ассистента (Aikicka)

## 1. Что делает и в чем польза (Ценность для продажи)
Это продвинутый **мультимодальный ИИ-ассистент** в Telegram, построенный на базе LangChain агента (n8n). 

**Ключевой функционал:**
*   **Мультиформатность:** Принимает текст, голосовые, фото и документы (PDF, Excel, Word), извлекает из них текст и анализирует.
*   **Синтез и распознавание речи:** Использует внешний Whisper для распознавания аудио и Edge TTS для ответа голосом.
*   **Генерация контента:** Умеет рисовать картинки (DALL-E 3 / Gemini) и писать песни через сервис Suno v5 (KIE API).
*   **Агентское поведение (Think):** У агента есть модуль стратегического планирования (Think), он оценивает запрос, строит шаги и выбирает нужные тулы.
*   **Работа с интернетом и файлами:** Ищет информацию через Perplexity, сохраняет файлы на Google Диск.
*   **Календарь:** Интеграция с Google Calendar для управления расписанием.
*   **Пейволл (Монетизация):** Встроена проверка по ID. Если пользователь чужой, бот просит \"Купить доступ у Валерия @Velkoshkin\".

**Как продавать (Упаковка пользы):**
Это не просто \"чат-бот\", это **\"Автономный цифровой сотрудник\"**. 
*   **Для бизнеса:** Экономит 2-3 часа в день на рутине (саммари встреч из аудио, парсинг документов, ресерч в интернете).
*   **Для криэйторов:** Автоматически генерирует музыку, изображения и идеи для контента.
*   **Фишка для продаж:** Демонстрация того, как он парсит голосовуху на 5 минут и выдает структурный план в Excel/Notion, закрывает сделки.

**Оценка стоимости:**
*   **B2B Установка (единоразово + настройка):** $1,500 - $3,000 (зависит от кастомизации под компанию).
*   **SaaS Подписка:** $15 - $49 / месяц (если прикрутить автоматический биллинг).

## 2. Как настроить (Деплой)
Чтобы развернуть этого агента:
1.  **Хостинг:** Установить n8n на VPS (например, Hetzner, Docker-compose).
2.  **Импорт:** Загрузить этот JSON в новый workflow n8n.
3.  **Креденды (Credentials):** Нужно создать и подключить в n8n следующие аккаунты:
    *   `Telegram Bot API` (создать бота через BotFather).
    *   `Google API` (Gemini, Google Drive OAuth2, Google Calendar).
    *   `PostgreSQL` (для памяти диалогов `n8n_nodes_langchain.memoryPostgresChat`).
4.  **Внешние микросервисы:** В схеме прописаны два внешних сервиса на Railway, их нужно поднять или заменить на готовое API:
    *   `openai-whisper-asr-webservice` (для транскрибации).
    *   `openai-edge-tts` (для озвучки).

## 3. Как улучшить (Стратегия масштабирования и монетизации)
С фокусом на генерацию выручки и надежность:

1.  **Supabase вместо голого Postgres (MUST HAVE):**
    *   Перенести память на Supabase.
    *   Добавить **Vector Store (pgvector) в Supabase** для RAG (чтобы бот знал базу знаний компании, регламенты, прайсы и мог продавать).
    *   Использовать Supabase Auth для управления подписками.
2.  **Автоматическая монетизация (Убрать ручные продажи):**
    *   Вместо сообщения \"Купи у Валерия\", интегрировать Stripe, CryptoPay или ЮKassa прямо в Telegram (через Payments API).
    *   Механика: Пользователь жмет `/buy`, оплачивает криптой/картой, webhook дергает n8n, меняет статус юзера в Supabase на `premium`, и доступ открывается автоматически. Это кратно повысит конверсию.
3.  **Интеграция CRM:**
    *   Подключить AmoCRM / Bitrix24. Бот должен квалифицировать лида и закидывать сделку в воронку.
4.  **Оптимизация расходов (Cost Effective):**
    *   Вместо тяжелых серверов на Railway для TTS/ASR, использовать `Groq API` (Whisper) — быстро и дешево.

## 4. API ключи, артефакты и лучшие ИИ Модели

**Необходимые API ключи:**
1.  **Telegram Bot API** (Бесплатно).
2.  **Google Cloud** — для Gemini (модели и зрение), Google OCR для документов.
3.  **KIE API Key** — для генерации музыки через Suno V5.
4.  **Perplexity API** — для глубокого поиска в интернете.
5.  **Supabase API Key & URL** — для базы данных и памяти.

**Рекомендуемые модели:**
*   **Оркестратор (Мозг Агента):** У вас уже прописан `gemini-2.5-flash-preview` — это **лучший вариант на рынке** для агентской логики по соотношению Скорость/Цена (очень дешево, быстро, огромное окно контекста).
*   **Для сложных когнитивных задач (Аналитика/Тексты):** Интегрировать `claude-3-5-sonnet` (Антропик) в параллельную ветку.
*   **Зрение:** `gemini-2.5-flash` справляется отлично.
*   **Голос продажника:** Интегрировать ключ `ElevenLabs`. Стоит чуть дороже, но вызывает **WOW-эффект** и конвертирует намного лучше, чем стандартый Edge TTS.
