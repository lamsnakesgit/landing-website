# Threads Auto Commenting & Posting Bot

## Статус: 🟡 БЭКЛОГ (ожидает ресерч)

---

## Идея
Автоматический постинг и комментинг в Threads (Instagram) для наращивания присутствия и привлечения клиентов.

## Архитектура (предварительная)

```
Google Sheets (контент-план)
    → Apps Script (таймер, HTTP-запрос)
    → Node.js API на ВПС (Express)
    → Playwright (управляет браузером)
    → Threads (постит/комментит)
```

## Стек
- Node.js + Express (API-сервер на ВПС)
- Playwright (управление браузером)
- Google Sheets + Apps Script (контент-план + триггер)
- Telegram бот (уведомления + одобрение перед постингом)

## Workflow одобрения
1. Агент подготавливает пост/комментарий
2. Отправляет в Telegram: «Запостить это? [Да/Нет]»
3. Ты нажимаешь «Да» → агент постит
4. Статус обновляется в Google Sheets
