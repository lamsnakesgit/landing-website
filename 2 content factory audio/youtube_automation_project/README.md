# Faceless YouTube Automation Project

This project contains the core templates and a detailed pipeline explanation for building a faceless YouTube automation system.

## Project Structure

- `templates.json`: Structured JSON templates for scripts, metadata, and conceptual n8n workflows.
- `PIPELINE.md`: A comprehensive "A to Z" guide on how the automation pipeline works, from idea generation to video upload.

## Getting Started

1. Read `PIPELINE.md` to understand the workflow.
2. Use the structures in `templates.json` as prompts for your AI tools (OpenAI, Claude, etc.).
3. Implement the logic in an automation tool like **n8n** or **Make.com**.
