# AI Automation Hub

Welcome to your centralized AI Automation workspace. This hub contains three specialized projects for YouTube automation, Lead Generation, and Google Cloud-based content production.

## 📂 Project Structure

### 1. [Google Content Factory](./docs/GOOGLE_STRATEGY.md)
- **Goal**: Maximize your 190€ Google Cloud budget.
- **Strategy**: Use Gemini 1.5 Flash and Imagen 3 to produce 300+ videos.
- **Key Files**: `docs/GOOGLE_STRATEGY.md`, `templates/google_templates.json`.

### 2. [YouTube Automation](./youtube_automation_project/)
- **Goal**: General faceless YouTube channel automation.
- **Strategy**: A-Z pipeline from idea to upload.
- **Key Files**: `PIPELINE.md`, `templates.json`.

### 3. [AI LeadGen Automation](./ai_leadgen_automation/)
- **Goal**: Automated B2B lead generation using AI Agents.
- **Strategy**: Sourcing, qualifying, and personalized outreach.
- **Key Files**: `LEADGEN_PIPELINE.md`, `leadgen_templates.json`.

---

## 🚀 Getting Started

1.  **Pick a Project**: Decide if you want to build a YouTube channel or a LeadGen system.
2.  **Read the Pipeline**: Each folder has a `.md` file explaining the "A to Z" workflow.
3.  **Use the Templates**: Copy the JSON structures into your automation tool (n8n, Make.com, or custom code).
4.  **Verify**: Follow the "Verification" steps in each project to ensure your automation is working correctly.
