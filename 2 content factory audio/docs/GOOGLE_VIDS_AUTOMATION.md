# Автоматизация Google Vids (docs.google.com/videos)

Google Vids — это новый инструмент в Google Workspace, который использует Gemini для создания видео. На данный момент (апрель 2026) **официального API для Google Vids не существует**, но есть способы его автоматизировать, используя тот факт, что это веб-приложение.

## Почему Google Vids?
- **Бесплатно**: Входит в подписку Google Workspace (Business/Enterprise).
- **Длительность**: До 30 минут в одном проекте.
- **AI-функции**: "Help me create" генерирует структуру, сценарий и подбирает стоковые видео/музыку.

---

## Способ 1: UI Автоматизация (Playwright / Puppeteer) — САМЫЙ РЕАЛЬНЫЙ
Так как API нет, единственный способ — имитация действий пользователя.

### Как это работает:
1. **Авторизация**: Используем `storage_state` в Playwright, чтобы не вводить логин/пароль каждый раз (Google блокирует ботов на странице логина).
2. **Создание**: Переходим на `docs.google.com/videos/create`.
3. **Промпт**: Вставляем текст в поле "Help me create".
4. **Генерация**: Ждем появления элементов сцен.
5. **Экспорт**: Нажимаем "File" -> "Download" -> "MP4".

### Пример скрипта (Python + Playwright):
Смотри актуальный файл `core/vids_automation.py`. Основные этапы:
1. **Авторизация**: Загрузка `google_auth.json`.
2. **Создание**: Переход на `docs.google.com/videos/create`.
3. **Промпт**: Ввод текста в Gemini.
4. **Генерация**: Ожидание кнопки "Create video".
5. **Экспорт**: Навигация через меню "File" -> "Download" -> "MP4".
6. **Рендеринг**: Ожидание завершения подготовки файла (может занять до 10 минут).

## Инструкция по запуску

1. **Установи зависимости**:
   ```bash
   pip install playwright
   playwright install chromium
   ```

2. **Сохрани сессию Google**:
   Запусти `save_auth.py`. Откроется браузер, войди в свой Google аккаунт вручную. После входа нажми Enter в терминале. Файл `templates/google_auth.json` будет создан.
   ```bash
   python scripts/save_auth.py
   ```

3. **Запусти автоматизацию**:
   Передай свой промпт как аргумент:
   ```bash
   python core/vids_automation.py "Создай видео про пользу ИИ для бизнеса"
   ```
   Скрипт создаст проект, дождется рендеринга и скачает готовый MP4 файл в текущую директорию.

## Способ 2: "Хак" через Google Drive API
Google Vids сохраняет проекты как файлы в Google Drive. 
- Можно попробовать **копировать** существующий "шаблонный" проект Vids через Drive API.
- Однако, на текущий момент Google Drive API не позволяет редактировать *содержимое* файла Vids (это проприетарный бинарный/JSON формат, как у Google Slides).

---

## Способ 3: Использование AppScript (Перспективно)
Google часто добавляет поддержку новых сервисов в AppScript.
- Проверь в редакторе AppScript, появился ли сервис `VidsApp` (по аналогии с `SlidesApp` или `DriveApp`).
- Если он появился, можно будет создавать видео одной строчкой кода: `Vids.create("My Video", prompt)`.

## ИТОГ:
Если нужно **максимально бесплатно** и именно в Google Vids:
1. Настрой **Playwright** с сохранением сессии (`auth.json`).
2. Напиши скрипт-прослойку, который берет сценарий из твоего n8n и "вбивает" его в браузер.
3. Это даст тебе доступ к бесплатным стокам Google и их движку рендеринга.

**Файлы для работы:**
- `templates/google_auth.json` — создай его, один раз зайдя в Google вручную через Playwright.
- `core/vids_automation.py` — основной скрипт автоматизации.
- `scripts/save_auth.py` — скрипт для первичной авторизации.
- `docs/VIDEO_ANALYSIS_PROMPT.md` — промпт для анализа референсов (например, твоего видео "sen sulu").

## Продвинутый Workflow: Воссоздание по референсу (1 в 1)

Если у тебя есть видео-референс (например, `IMG_1400_sen sulu_beauty_.MP4`) и ты хочешь сделать такое же в Google Vids:

1. **Анализ**:
   - Загрузи свое видео в **Gemini 1.5 Pro** (через Google AI Studio или интерфейс Gemini).
   - Используй промпт из файла `VIDEO_ANALYSIS_PROMPT.md`.
   - Gemini выдаст тебе детальный сценарий и **Мастер-промпт для Google Vids**.

2. **Автоматизация**:
   - Скопируй полученный Мастер-промпт.
   - Запусти скрипт:
     ```bash
     python core/vids_automation.py "ВСТАВЬ_ПОЛУЧЕННЫЙ_ПРОМПТ_ЗДЕСЬ"
     ```

3. **Результат**:
   - Скрипт зайдет в Google Vids, вставит промпт, Gemini внутри Vids создаст структуру сцен, максимально похожую на оригинал, и скачает готовый файл.

## Советы по промптам для Google Veo (внутри Vids)

Google Veo лучше всего понимает кинематографические термины. Добавляй их в промпты для отдельных сцен:
- **Движение**: `Cinematic slow pan`, `Drone shot`, `Handheld shaky cam` (для динамики).
- **Освещение**: `Golden hour lighting`, `Soft studio rim light`, `Cyberpunk neon glow`.
- **Качество**: `8k resolution`, `highly detailed textures`, `photorealistic`.
- **Стиль**: `Film noir`, `Wes Anderson aesthetic`, `National Geographic style`.
