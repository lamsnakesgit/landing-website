# Стратегия: Камчатка / Природа (Longs & Shorts)

Эта стратегия описывает, как из одного большого 10-минутного ролика про Камчатку сделать 10-15 виральных Shorts, используя n8n и Google Cloud.

## 1. Две ветки в n8n (Branching)

### Ветка 1: Long-form (10 минут)
- **Формат**: 16:9 (1920x1080).
- **Цель**: Глубокое погружение, удержание аудитории, монетизация через RPM.
- **Структура**: 
    - Вступление (Завораживающие кадры вулканов).
    - 5-7 глав (Долина Гейзеров, Медведи, Курильское озеро и т.д.).
    - Заключение.
- **Генерация**: Batch API (Gemini 1.5 Flash для сценария на 1500 слов).

### Ветка 2: Shorts (60 секунд)
- **Формат**: 9:16 (1080x1920).
- **Цель**: Охваты, подписки, "крючки" для перехода на длинное видео.
- **Типы Shorts**:
    - "Топ 3 факта о вулканах".
    - "Самое опасное место Камчатки".
    - "Как выжить при встрече с медведем".
- **Генерация**: n8n берет *одну главу* из лонга и переписывает её под формат Shorts (быстрый темп, яркий заголовок).

---

## 2. Логика "Умной нарезки" (Smart Clipping)

В n8n мы настраиваем узел **AI Agent**, который делает следующее:
1. Получает полный сценарий лонга.
2. Ищет "High-Intensity Moments" (самые интересные факты).
3. Генерирует для каждого момента:
    - Вертикальный `image_prompt` для Imagen 3.
    - Динамичный `audio_text` (на 45-50 секунд).
    - Заголовок-крючок (Hook).

---

## 3. Промпты для Imagen 3 (Nature/Cinematic)

Для Камчатки важно передать масштаб и атмосферу. Используем ключевые слова:
- *'Cinematic aerial shot, 8k, National Geographic style, volcanic smoke, turquoise water, realistic textures, soft morning light, wide angle lens.'*

---

## 4. Как не делать "ахинею" на 10 минутах?

Чтобы 10-минутный ролик не превратился в кашу:
1. **Контрольные точки**: Gemini должна выдавать сценарий по главам. Каждая глава — это отдельный JSON-объект.
2. **Сквозной стиль**: В каждом промпте для картинок должен быть один и тот же `style_reference` (например, "Cinematic Nature Photography").
3. **Музыка**: В лонге музыка должна меняться (спокойная в начале, эпичная на вулканах). В Shorts — всегда динамичная.
