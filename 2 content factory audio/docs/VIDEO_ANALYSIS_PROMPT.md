# Prompt for 1:1 Video Analysis and Recreation (Google Veo / Google Vids)

Use this prompt with a multimodal AI (Gemini 1.5 Pro or GPT-4o) by uploading your reference video.

---

## The Prompt (English)

**Task**: Analyze the attached video and create a detailed, scene-by-scene technical script for recreation in Google Vids or Google Veo.

**Instructions**:
1. **Visual Style**: Describe the overall aesthetic (lighting, color palette, camera movement, lens type).
2. **Scene Breakdown**: For every scene/cut, provide:
    - **Timestamp**: Start and end.
    - **Visual Description**: Exactly what is happening (subject, action, background).
    - **Camera Work**: (e.g., "Slow zoom in", "Panning left", "Static close-up").
    - **Text Overlay**: Any text appearing on screen (font style, position).
    - **Audio/Voiceover**: Transcribe any speech or describe the background music/SFX.
3. **Google Vids Prompt Generation**: Based on the analysis, generate a single, comprehensive prompt (max 2000 characters) that I can paste into the "Help me create" box in Google Vids to recreate this video's structure and vibe.
4. **Google Veo Prompt Generation**: Generate 3-5 specific prompts for Google Veo to recreate the key cinematic shots from this video.

**Output Format**:
- **Summary of Style**
- **Table of Scenes**
- **Master Prompt for Google Vids**
- **Cinematic Prompts for Google Veo**

---

## Промпт (Russian)

**Задача**: Проанализируй прикрепленное видео и создай детальный технический сценарий для воссоздания 1 в 1 в Google Vids или Google Veo.

**Инструкции**:
1. **Визуальный стиль**: Опиши общую эстетику (освещение, цветовая палитра, движение камеры, тип линзы).
2. **Разбор по сценам**: Для каждой сцены/склейки укажи:
    - **Таймкод**: Начало и конец.
    - **Визуальное описание**: Что именно происходит (объект, действие, фон).
    - **Работа камеры**: (например, "Медленный зум", "Панорама влево", "Статичный крупный план").
    - **Текст на экране**: Любой текст, его стиль и позиция.
    - **Аудио/Озвучка**: Транскрипция речи или описание фоновой музыки/эффектов.
3. **Промпт для Google Vids**: На основе анализа создай один большой промпт (до 2000 символов), который я вставлю в поле "Help me create" в Google Vids, чтобы воссоздать структуру и атмосферу этого видео.
4. **Промпты для Google Veo**: Создай 3-5 специфичных промптов для Google Veo, чтобы перегенерировать ключевые кинематографичные кадры из этого видео.

**Формат вывода**:
- **Резюме стиля**
- **Таблица сцен**
- **Мастер-промпт для Google Vids**
- **Кинематографичные промпты для Google Veo**
