# AI Lead Generation Automation Project

This project contains the core templates and a detailed pipeline explanation for building an AI-driven Lead Generation system using AI Agents.

## Project Structure

- `leadgen_templates.json`: Structured JSON templates for AI Agent prompts, qualification criteria, and outreach messages.
- `LEADGEN_PIPELINE.md`: A comprehensive guide on how the lead generation pipeline works, from sourcing to CRM logging.

## Getting Started

1. Read `LEADGEN_PIPELINE.md` to understand the workflow.
2. Use the prompts in `leadgen_templates.json` to configure your AI Agents.
3. Implement the logic in an automation tool like **n8n** using the **AI Agent Node**.
